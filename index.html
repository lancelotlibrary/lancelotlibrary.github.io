About LANCELOT - 5min ⏱️
 

Access terabytes of data with one click

N-body simulations are known to produce terabytes or even petabytes of data, which makes it hard to download and analyse, post-process the results.

During the process of LANCELOT development, we addressed this issue and created remote JupyterLab server, which can be accessed freely without any authentification in read-only mode.

This means that you can analyse terabytes of simulation data (matter power spectrum, halo catalogues, lightcone projections etc.) using pre-built Python libraries like Pylians3 and yt-project.

However, it is also possible to download data partially or everything at once using curl command generator in the documentation.

No need to download data

It is possible to access and plot all of the post-processing data via SSH protocol and JupyterLab.

Broad Aims

We probed many modified gravity theories and apart from that included massive neutrinos as well as non-minimally coupled Lagrangian.

Fully open-source

Each data product can be accessed freely, from N-body code parameter files to matter power spectra and convergence maps.
